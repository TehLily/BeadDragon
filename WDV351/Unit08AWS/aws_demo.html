Cloud Computing

Cloud computing uses the internet to provide computing resources like servers, apps, storage, dev tools, networking, etc from a remote data center. A Cloud Service Provider (CSP) like Amazon Web Services (AWS), Microsoft Azure, Google Cloud, or IBM Cloud manages the data center and provides resources either based on a monthly subscription or on a usage basis. This can help lower costs spent on IT, have faster implementation, and improves ability to scale.

There are several models of cloud services that can be provided.

Software as a Service (SaaS)
Delivers software to users and supports all application aspects, except user data
Examples: SalesForce, Blackboard, DocuSign
Infrastructure as a Service (IaaS)
Provides access to resources like servers, networking, storage, and virtualization to users on demand
Examples: AWS, Rackspace, Google Compute Engine (GCE)
Platform as a Service (PaaS)
Provides the complete platform for development, implementation, and management of applications
Examples: Java runtimes, mySQL, Tomcat server
Backend or Mobile Backend as a Service (BaaS/MBaaS)
Provides web and mobile app backend support like user management, social media integration, and notifications
Examples: Amplify on AWS, Firebase on Google Cloud, Backendless
